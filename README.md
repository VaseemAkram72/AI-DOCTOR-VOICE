🩺 AI Doctor Voice Assistant Project

An intelligent AI Doctor Voice Assistant developed using Python, LLMs (Large Language Models), and the Groq API that can understand user voice input, process medical-related queries, and generate smart voice-based responses in real time.

This project combines the power of Artificial Intelligence, Voice Processing, and Generative AI to create an interactive healthcare assistant capable of responding within seconds.

⚡ Role of LLM & Groq API

This project utilizes LLMs (Large Language Models) integrated with the Groq API to process and solve user queries within seconds.

The AI system helps in:

Understanding voice commands
Processing medical-related questions
Generating intelligent responses
Delivering fast and accurate AI interaction
Enhancing real-time communication experience

The Groq API enables ultra-fast AI inference, making the voice assistant highly responsive and efficient.
